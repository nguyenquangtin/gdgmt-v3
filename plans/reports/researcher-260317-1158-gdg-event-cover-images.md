## Image Download Notes

### Download Recommendations

**For web use (optimized):** Use URLs as-is with `dpr_2.0` parameter
**For local storage/archive:** Replace `dpr_2.0` with `dpr_1.0` for full resolution without device-specific scaling

**Cloudinary URLs:**
- Automatically optimized for web delivery
- Support format negotiation (jpg/png/webp)
- Can modify parameters: `w_1200`, `h_630` for resizing
- Replace `f_auto` with `f_jpg` or `f_png` to force format

**Luma CDN URLs:**
- Direct image path available without CDN wrapper
- Supports standard image optimization parameters
